# Learning Platform Backend API

This is the backend API for the Learning Management System (LMS). It handles user authentication and course management operations... By Zaara

## Tech Stack
- **Node.js**: Runtime environment
- **Express.js**: Web framework
- **MongoDB**: Database
- **JWT**: For secure authentication

## Features
- **User Auth**: Sign up and Login (returns JWT Token).
- **Courses**:
  - Create new courses (Admin/Teacher).
  - View all courses.
  - View single course details.
  - Update and Delete course functionality.
